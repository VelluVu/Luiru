# Luiru — Liero Clone

> A C# recreation of Liero, the classic real-time worm shooter, built in Unity as a school project.

![Unity](https://img.shields.io/badge/Engine-Unity-black?logo=unity) ![Language](https://img.shields.io/badge/Language-C%23-purple) ![Type](https://img.shields.io/badge/Type-School%20Project-orange) ⭐ 1 star

## About

**Luiru** is a Unity-based clone of **Liero** — the legendary Finnish real-time worm shooter game where two players dig through destructible terrain and shoot each other with an arsenal of weapons. This project focuses on destructible terrain simulation, weapon variety, and two-player local multiplayer.

> Fun fact: Liero itself was created by a Finnish developer, making this a fitting project for a Finnish game programming student!

## Features

- 💣 Destructible terrain using pixel/texture manipulation
- 🪱 Worm-style player movement through terrain
- 🔫 Multiple weapon types with different projectile behaviours
- 👥 Two-player local multiplayer (split controls)
- 💥 Explosion physics and terrain deformation
- ❤️ Health system and death/respawn

## Built With

- **Unity** (2D)
- **C#** — terrain destruction, weapons, and player control

## Getting Started

### Prerequisites

- Unity Editor (2019.x or later recommended)

### Running the Project

1. Clone the repository:
   ```bash
   git clone https://github.com/VelluVu/Luiru.git
   ```
2. Open in Unity Hub.
3. Open the main scene from `Assets/Scenes/`.
4. Press **Play** — Player 1 and Player 2 use separate keyboard controls.

## Context

Cloning Liero is a technically demanding school project — destructible terrain requires real-time texture modification, and the game's fast-paced physics demand careful performance tuning. One of the more ambitious early school projects.
